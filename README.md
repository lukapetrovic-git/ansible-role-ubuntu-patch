# ansible-role-ubuntu-patch
Ansible Role for patching Ubuntu Servers

# Requirements
- Ansible 2.10 or later
- Python version

# Tested on
- Ubuntu 18.04 LTS
- Ubuntu 20.04 LTS
- Ubuntu 22.04 LTS
- Ubuntu 24.04 LTS

# Role Variables
All settable variables with explanations and links are located in the defaults/main.yml
